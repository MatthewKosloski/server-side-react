# server-side-react
An extremely simple server-side React example with Redux and React Router.

## Dependencies
- To install the dependencies, run `npm install`.
- Nodemon needs to be installed globally, `npm install -g nodemon`.

## Development
```
  $ npm run dev
```

## Production
```
  $ npm start
```

## Test

```
  $ npm run test
```

```
  $ npm run test:watch
```
